# PVE Scripts Local

Web-based management interface for Proxmox VE community helper scripts. Discover, download, and execute scripts locally with real-time terminal output — no curl|bash needed.

## Overview

| Property | Value |
|----------|-------|
| Node | TMG (tmg) |
| LXC | 116 |
| IP | 10.0.0.152 |
| Tailscale IP | 100.121.139.111 |
| Port | 3000 |
| Storage | local-zfs |
| Status | ✅ Active |

## Access

| Method | URL |
|--------|-----|
| Tailscale HTTPS | https://pve-scripts-local.tail-scale.ts.net |

## Installation

Deployed via community helper script. Tailscale added afterward using `ts-init` from the quickref command reference.
