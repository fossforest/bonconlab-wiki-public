## Purpose

Browser-based toolbox for common dev/sysadmin tasks, including:

- Encoding/decoding (Base64, URL, JWT, etc.)
- Network utilities (CIDR calculator, DNS lookup, etc.)
- Formatters (JSON, YAML, SQL, etc.)
- Generators (UUID, password, hash, etc.)

No authentication required — access is restricted by Tailscale network membership.
