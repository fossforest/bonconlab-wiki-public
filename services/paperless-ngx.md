# Paperless-ngx

Document management system with built-in OCR. Scans, indexes, and archives documents with full-text search.

## Features

- Full-text search across all archived documents
- Automatic tagging, correspondent, and document type classification
- Tesseract OCR built in — no separate dependency required
- iOS scanning via mobile browser

## Storage

Documents stored on sb-1tb-ssd inside the container.
